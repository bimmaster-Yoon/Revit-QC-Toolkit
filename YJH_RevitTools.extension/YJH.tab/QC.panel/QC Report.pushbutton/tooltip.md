# Revit QC Report

현재 Revit 문서의 Sheet, View 및 주요 Shared Parameter 입력 상태를 검사합니다.

- Sheet 번호, 이름 및 View 배치 상태 검사
- View 이름, 임시 키워드, 축척, View Template 및 Sheet 배치 상태 검사
- Room과 주요 설비 카테고리의 Shared Parameter 입력 상태 검사
- pyRevit 출력창에 Summary와 상세 Issue List 표시
- QC 결과 CSV 내보내기

이 도구는 읽기 전용으로 실행되며 Revit 모델을 수정하지 않습니다.
