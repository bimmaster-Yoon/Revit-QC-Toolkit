# QC Toolkit Icon Style

- Output: 64 × 64 px transparent PNG
- Master drawing: 512 × 512 px, 8× supersampling
- Dark navy: `#14253D`
- Orange: `#F47C20`
- White shape fill with transparent outer background
- Thin, rounded lines for visibility on the Revit ribbon
- No external product or AI service logos

Run `generate_toolkit_icons.py` with Python and Pillow to regenerate all five button icons.
