# Scan QC

Point Cloud 기반 Revit 모델 오차 검토

검토 범위와 Point Cloud를 선택하고 Scan QC 작업용 View 생성 조건을 설정합니다.

- 선택된 Wall 개수 확인
- Active Plan Level 또는 Selected Walls 검토 범위 선택
- 사용할 Point Cloud Instance 선택
- 기본 허용오차 기준 확인
- QC Plan/3D 작업 View 및 보고서 옵션 선택

현재 단계에서는 Standards 확인/설치, QC 작업 View, 2D Revision Cloud/TextNote Preview를 생성합니다.
3D Preview 표현은 비활성화되어 있습니다.
실제 편차 계산, PDF/CSV 생성, Point Cloud 색상 변경은 수행하지 않습니다.
